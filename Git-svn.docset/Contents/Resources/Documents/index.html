Git-svn

Cloning & Setup

Clone a Subversion repository as a Git repository using standard Subversion layout

git svn clone -s --prefix=origin/ <subversion-url>

Example: git svn clone -s --prefix=origin/ svn://example.com/repository/MyProject

Recursively find all the svn:ignore properties and generate a matching .gitignore file

git svn create-ignore

Recursively find all the svn:ignore properties and append the output to $GIT_DIR/info/exclude

git svn show-ignore >> .git/info/exclude

Local Changes

List changed files in your working directory

git status

List changes to tracked files

git diff

Stage an untracked file for the next commit

git add <file>

Stage all changes for the next commit

git add .

Note: This will also add untracked files to the index.

Commit previously staged changes

git commit

Commit all local changes in tracked files

git commit -a

Note: This will not include untracked files. Use git add . or git add <file> to stage untracked files.

Commit History

Show all commits

git log

Show changes over time for a specific file

git log -p <file>

Who changed what and when in a specific file

git blame <file>

Branches

List all local branches

git branch

List all remote branches

git branch -r

Note: This lists all branches ever created in the Subversion repository (including deleted ones).

Switch HEAD branch

git checkout <branch>

Create a new branch based on your current HEAD

git branch <new-branch>

Create a new tracking branch based on a remote branch

git branch --track <new-branch> <remote-branch>

Shortcut for creating a new remote tracking branch and switching to it

git checkout -b <new-branch> <remote-branch>

Note: This assumes the Git default configuration of --track for remote tracking branches.

Delete a local branch

git branch -d <branch>

Note: Use git branch -D <branch> to delete a branch regardless of merge status.

Getting & Publishing Updates

Download all changes from the Subversion remote

git svn fetch

Rebase you local tracking branch to synchronize with the Subversion remote branch

git svn rebase

Publish your local commits to the Subversion remote

git svn dcommit
